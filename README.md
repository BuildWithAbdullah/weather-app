# Weather App

A responsive and intuitive weather application built using HTML, CSS, and JavaScript. The app uses the OpenWeatherMap API to fetch and display real-time weather data, including temperature, weather condition, and local time.

## Features
- Fetches live weather data for any city using the OpenWeatherMap API.
- Displays the current temperature, weather condition, and local time.
- Shows the day of the week and the exact date.
- Clean, user-friendly interface with responsive design.

## Technologies Used
- **HTML**: Structure and markup.
- **CSS**: Styling and responsive layout.
- **JavaScript**: Fetch API to get weather data.
- **OpenWeatherMap API**: For fetching weather information.
